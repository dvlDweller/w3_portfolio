# AGENTS.md

Purpose: concise guidance for AI coding agents working on this repository.

Project
- Static personal CV website (HTML + CSS).
- Key files: `index.html`, `style.css`.

Run / Build
- No build step. Open `index.html` in a browser to view locally.

Tests
- None.

Conventions for agents
- Make minimal, focused edits; prefer small patches using apply_patch.
- Do not add dependencies or tooling without explicit approval.
- If a change affects layout or content, preview by opening `index.html`.
- Ask before creating new top-level files or modifying `.vscode/` settings.

Notes for maintainers
- This repo has no CI or build tools. Keep guidance short and link to docs if added.
